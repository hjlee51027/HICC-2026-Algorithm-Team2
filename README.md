## 📌 스터디 운영 및 인증 방법

### 1. 폴더 구조화
깃허브 레포지토리 내에서 각자 **자기 이름으로 된 폴더**를 생성하여 코드를 관리합니다.

- 권장 구조: [이름]/[주차]/[실습파일]   (예: lee_sunwoo/Week1/실습1.java)
---

### 2. 코드 리뷰
본인의 코드를 Push한 후, 팀원들의 코드를 확인하며 **리뷰를 주고받습니다.**
